CodeGenius - 一个会把“思考失误”也给你看的 AI 编程助手

它现在很笨，但它**绝不撒谎、绝不隐藏**。

## 核心亮点（其他所有 Agent 都不敢给你的）

> 你会亲眼看到它完整犯错、纠错、找到正确文件的全过程：

\`\`\`text
用户：把工具类里的 redis 连接改成连接池

Agent 思考：工具类可能是 utils/redis.py → read_file("utils/redis.py") → 文件不存在
Agent 思考：那可能是 src/common/redis_client.py → list_dir("src") → 看到 common 目录
Agent 思考：再 list_dir("src/common") → 发现 redis_client.py → 读取 → 找到代码 → 修改 → 写回
\`\`\`

整个过程就像一个实习生在你面前现场百度 + 试错，你看得见每一步对错。

这才是 2025 年最稀缺的能力：**可观察、可信任、可介入**。

## 当前真实能力（2025-11-18）

- 只能指定 1 个项目文件夹
- 只有 4 个工具：list_dir、read_file、write_file（直接覆盖）、delete_file
- 没有预览、没有确认、没有备份
- 但它会在聊天框里把每一次“猜错 → 再试”的过程完整展示给你

## 真实使用体验（90% 用户看完都会说“卧槽，这也太真实了”）

\`\`\`
你：把所有用了 print 的地方改成 logger

Agent：
1. 先 list_dir(".") 看看项目结构
2. 发现 src/ 和 app/ 两个目录
3. 先读 src/main.py → 有 print → 修改并写回
4. 再读 app/utils.py → 也有 → 修改写回
5. 最后告诉你：共修改了 7 处
\`\`\`

你全程知道它看了哪些文件、改了哪些行，零黑箱。

## 和主流工具的终极差异

| 项目                   | CodeGenius（现在）                    | Cursor / Claude Code / Aider |
